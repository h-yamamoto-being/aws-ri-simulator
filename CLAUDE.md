# CLAUDE.md

## プロジェクト概要

AWS RDS for Oracle のリザーブドインスタンス（RI）とオンデマンドの年間コストを、
ユーザーが自由に定義した複数シナリオで同時比較するWebアプリ。

---

## 技術スタック

- **フレームワーク**：Vue 3 + TypeScript + Vite
- **スタイル**：Tailwind CSS
- **グラフ**：Chart.js + vue-chartjs
- **バックエンド**：なし（フロントエンドのみ）
- **データ永続化**：なし

---

## ビジネス仕様

### 料金プランの種類（4パターン）

| プランタイプ | 割引率 | 入力方法 | 支払い方法 |
|-------------|--------|---------|-----------|
| オンデマンド | なし | 時間単価（USD/hour）を入力 → 月額を自動計算 | 毎月その月の為替レートで円換算 |
| RI・前払いなし | 28%割安 | 時間単価から自動計算 | 毎月その月の為替レートで円換算 |
| RI・一部前払い | 30%割安 | AWSページから前払い額・月額を転記入力※ | 前払い分は1ヶ月目のレートで円換算。月額分は毎月その月のレートで円換算 |
| RI・全額前払い | 32%割安 | 時間単価から自動計算 | 年間一括を1ヶ月目のレートで円換算。2〜12ヶ月目は0円 |

※一部前払いの前払い額と月額の比率はAWSがインスタンスタイプごとに固定しているため、ユーザーはAWSページから実額を転記する。

### 計算式

```
onDemandMonthly_USD = hourlyRate × hoursPerMonth
riNoUpfront_USD     = onDemandMonthly_USD × 0.72   // 月額（28%割引）
riFullUpfront_USD   = onDemandMonthly_USD × 12 × 0.68  // 年間一括（32%割引）
```

一部前払いの前払い額・月額はユーザー入力値をそのまま使用する。

### 月次支払い（円換算）

- オンデマンド：`onDemandMonthly_USD × rates[month]`（毎月）
- RI前払いなし：`riNoUpfront_USD × rates[month]`（毎月）
- RI一部前払い：1ヶ月目は `(前払い額 + 月額) × rates[0]`、2ヶ月目以降は `月額 × rates[month]`
- RI全額前払い：1ヶ月目のみ `riFullUpfront_USD × rates[0]`、2〜12ヶ月目は0円

### 為替レート

- **全シナリオ共通の1セット**（12ヶ月分）。シナリオごとには持たない。
- 画面上部に独立したセクションとして配置する。
- デフォルト150円/ドル。一括設定（全月に同じレートを適用）と月別個別設定の両方をサポート。
- 計算時のレートの使い方はプランによって以下のとおり：
  - オンデマンド・RI前払いなし・RI一部前払いの月次分：毎月のレートをそれぞれ使用
  - RI一部前払いの前払い分・RI全額前払い：1ヶ月目のレートのみ使用

## コアコンセプト：シナリオ比較

ユーザーは「シナリオ」を自由に追加・削除・編集できる（最大8件）。
各シナリオは独立してプランタイプ・設定値・名前・色を持つ。
為替レートは全シナリオ共通。
全シナリオの結果をグラフ・表で常時同時表示する。

初期表示は上記4プランを1シナリオずつ、デフォルト為替150円で表示する。
デフォルトの時間単価は `8.896 USD/hour`（db.r6i.4xlarge 東京リージョン・マルチAZ・ライセンス込み）。
一部前払いのデフォルト値は前払い `$27,275`・月額 `$2,272.93`（同インスタンスの実額）。

## ファイル構成

```
src/
  App.vue
  types.ts
  constants.ts
  utils/
    calculator.ts         // 計算ロジック（純粋関数）
    scenarioHelpers.ts    // シナリオ追加・削除・更新
  composables/
    useSimulator.ts       // scenarios・rates・results のstate管理
  components/
    ScenarioList.vue      // シナリオカード一覧・追加ボタン
    ScenarioCard.vue      // 個別シナリオカード（プランタイプ別フィールド切替）
    ExchangeRateInput.vue // 為替レート入力（一括＋月別）・全シナリオ共通セクションとして画面上部に配置
    ResultSummary.vue     // サマリーカード（全シナリオ）
    ResultChart.vue       // 累計支払額の折れ線グラフ（vue-chartjs）
    ResultTable.vue       // 月別比較表（横スクロール対応）
    BreakEvenNote.vue     // 損益分岐点テキスト
```

## UI要件

- レスポンシブ対応（スマホ〜PC）
- リアルタイム計算（入力変更のたびに即時更新）
- 数値表示：円は小数点なし・3桁カンマ、ドルは小数点2桁
- ScenarioCard はプランタイプに応じて入力フィールドを切り替える
  - オンデマンド・前払いなし・全額前払い：時間単価・月間稼働時間
  - 一部前払い：前払い額（USD）・月額（USD）。ラベルはAWSページの表記（「RI の前払い料金」「RI の月額料金」）に合わせる
- シナリオが1件のみの場合は削除ボタンを無効化
